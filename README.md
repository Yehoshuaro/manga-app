# Manga App  

A simple manga reading app built with Flutter and Node.js

## Features  
- Browse a list of manga  
- View manga details  
- Read manga chapters  

## Technologies Used  
### Backend:  
- Node.js + Express  
- MongoDB + Mongoose  
- JWT for authentication  

### Frontend:  
- Flutter (Dart)  
- Provider for state management  
- Dio for API requests
